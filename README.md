Responsive WooCommerce product cards with AJAX Quick Add, shortcode manager, and live cart updates.

== Description ==

**Product View Card** is a lightweight WooCommerce plugin that displays products in clean, responsive card layouts with AJAX-powered Quick Add functionality.

It is ideal for:
- Landing pages
- Product highlights
- Custom shop sections
- Elementor / Gutenberg pages
